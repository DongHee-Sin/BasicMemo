## 주요기능
### 메모 작성
* 첫번째 개행 문자를 기준으로 타이틀과 컨텐츠를 구분하여 저장
* 내용이 입력되지 않으면 저장하지 않거나 삭제
* 작성된 메모 공유 가능

### 메모 목록
* 메모 삭제 기능
* 메모 고정 기능
   * 최대 5개 제한
* 메모 검색 기능
   * 입력되는 문자를 실시간으로 검색
   * 검색 키워드에 해당하는 단어의 텍스트 컬러 변경
